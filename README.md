##  ✨Project Overview

인스타그램, 트위터를 참고하여 만들어 본 간단한 SNS 프로젝트
User, Contents, Quote 데이터는 `dummyjson.com/` 에서 가져와 사용하였습니다.
이미지는 `picsum.photos`, `source.unsplash.com`, `i.pravatar.cc` 에서 가져와 사용하였습니다.

> [데모 링크 바로가기](https://swssbw.github.io/shop/)

### **사용기술**

- JavaScript (ES6+)
- TypeScript
- Next.js
- Redux, Redux-toolkit

### **구현기능**

- 랜딩페이지 콘텐츠 무한스크롤
- 키워드 검색(추천 키워드 : love)
- 유저페이지
